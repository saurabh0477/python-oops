# 🐍 Python OOPs

A complete beginner-friendly repository for learning **Object-Oriented Programming (OOP)** in Python with simple explanations, clean code examples, and practice programs.

This repository covers core OOP concepts step by step to help students and beginners build strong programming fundamentals in Python.

---

## 📚 Topics Covered

- ✅ Classes and Objects
- ✅ Constructors (`__init__`)
- ✅ Instance Variables and Methods
- ✅ Class Variables and Class Methods
- ✅ Static Methods
- ✅ Inheritance
- ✅ Method Overriding
- ✅ Polymorphism
- ✅ Encapsulation
- ✅ Abstraction
- ✅ Operator Overloading
- ✅ File Handling using OOP
- ✅ Practice Programs
- ✅ Mini Projects

---

## 🛠️ Technologies Used

- Python 3

## 🚀 Getting Started

### Clone the Repository

```bash

```

### Navigate to the Project Folder

```bash
cd python-oops
```

### Run Python Files

```bash
python filename.py
```
